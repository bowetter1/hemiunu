// Under the hood

Real Infrastructure,
Not Abstractions

Each project gets an ephemeral Linux environment powered by Daytona.io. Install packages, run databases, execute arbitrary commands. The AI works in the same environment a developer would.

Apex runs on Railway with a FastAPI backend and PostgreSQL. Sandboxes run on Daytona Cloud. The AI orchestrates everything through the Daytona SDK.

# What happens when you create a project sandbox = client.create(CreateSandboxFromImageParams( image="python:3.12-slim-bookworm", name="apex-my-project", )) # Full filesystem access sandbox.fs.upload_file( html.encode("utf-8"), "/workspace/public/index.html", ) # Run any command result = sandbox.process.exec( "pip install fastapi uvicorn" ) # Install databases sandbox.process.exec( "apt-get install -y postgresql" ) # Get live preview URL preview = sandbox.get_preview_link(8000) # https://8000-abc123.proxy.daytona.works
